# Ship_Godot
This project provides a basic idea on how to control a ship (which is in Godot) using c++

  * PI controller implementation 
    * eigen3 library
    
  * Kalman Filter 
    * eigen3 library
    
  * IPC using Sockets
    * Multiple Client- Server
    
  * Multithreaded Socket
    * Mutex locking
